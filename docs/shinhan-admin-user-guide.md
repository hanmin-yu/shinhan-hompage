# 신한관세법인 홈페이지 관리자 사용 가이드

이 문서는 홈페이지 내용을 직접 수정하는 운영자를 위한 안내서입니다. 개발 지식이 없어도 로그인, 문구 수정, 구성원 관리, 뉴스/소식지 등록을 할 수 있도록 화면 기준으로 설명합니다.

## 1. 관리자 화면 접속

관리자 화면 주소:

```text
/admin/login
```

운영 서버 도메인이 `https://www.example.com`이라면 아래처럼 접속합니다.

```text
https://www.example.com/admin/login
```

로그인 화면:

![관리자 로그인 화면](assets/admin-login.png)

로그인 순서:

1. `아이디`에 관리자 ID를 입력합니다.
2. `비밀번호`에 관리자 비밀번호를 입력합니다.
3. `관리자 로그인` 버튼을 누릅니다.
4. 로그인에 성공하면 `홈` 관리자 화면으로 이동합니다.

로그인 후 처음 보이는 화면:

```text
/admin/content/home
```

로그인이 안 될 때 확인할 내용:

- 아이디와 비밀번호 앞뒤에 공백이 들어가지 않았는지 확인합니다.
- 대문자/소문자 입력이 맞는지 확인합니다.
- 여러 번 실패하면 운영 담당자에게 비밀번호 변경 여부를 확인합니다.

## 2. 관리자 화면 공통 사용법

관리자 화면 상단에는 주요 관리 메뉴가 표시됩니다.

| 메뉴 | 수정할 수 있는 내용 |
| --- | --- |
| 홈 | 메인 화면 슬라이드, 무역 동향/소식지/사무소 섹션 문구 |
| 회사소개 | 회사소개, 연혁, 인사말 관련 문구 |
| 구성원 | 임원진, 전문가, 자문위원 등 구성원 정보와 사진 |
| 업무분야 | 수출입통관, FTA, AEO 등 업무분야 페이지 문구 |
| IT | IT 페이지 문구와 서비스 소개 |
| 소식/자료 | 신한 NEWS, 소식지, 신한 Insights |
| 문의 | 문의 페이지, 부정행위 접수창구 문구 |
| 채용 | 채용 페이지 문구, 복리후생, 채용 링크 |
| 오시는 길 | 서울본사 주소, 연락처, 지도 안내 문구 |

공통 버튼:

| 버튼 | 의미 |
| --- | --- |
| 저장 | 현재 화면에서 수정한 내용을 홈페이지에 반영합니다. |
| 되돌리기 | 마지막으로 저장된 내용으로 되돌립니다. |
| 로그아웃 | 관리자 화면에서 나갑니다. |
| 새 탭에서 크게 보기 | 실제 홈페이지 화면을 새 탭으로 열어 확인합니다. |

중요한 사용 원칙:

- 내용을 입력한 뒤 반드시 `저장` 버튼을 눌러야 반영됩니다.
- 저장 전에 다른 메뉴로 이동하면 작성 중인 내용이 사라질 수 있습니다.
- 저장 후 실제 홈페이지에서 새로고침하여 반영 여부를 확인합니다.
- 한글 문구와 영문 문구가 따로 있는 항목은 둘 다 수정합니다.

## 3. 운영 모드와 읽기 전용 모드

관리자 화면 상단에는 현재 상태가 표시됩니다.

| 표시 | 의미 |
| --- | --- |
| 운영 모드 | 저장, 삭제, 업로드가 가능한 실제 운영 상태입니다. |
| 데모 읽기 전용 | 화면 확인만 가능하고 저장은 되지 않는 상태입니다. |

`데모 읽기 전용`으로 표시되면 내용을 수정해도 저장되지 않습니다. 실제 운영 반영이 필요하면 서버가 운영 모드로 실행되어 있는지 운영 담당자에게 확인해야 합니다.

## 4. 홈 화면 수정

접속 경로:

```text
/admin/content/home
```

홈 메뉴에서는 메인 화면과 주요 섹션 문구를 수정합니다.

### 4.1 메인 비주얼 수정

`메인 비주얼` 탭에서는 홈페이지 첫 화면의 슬라이드를 수정합니다.

주요 입력 항목:

| 항목 | 설명 |
| --- | --- |
| 관리용 이름 | 관리자 화면에서 구분하기 위한 이름입니다. |
| 상단 문구 | 메인 제목 위에 작게 표시되는 문구입니다. |
| 메인 문구 | 큰 제목입니다. |
| 요약 문구 | 제목 아래 설명 문구입니다. |
| 이미지 경로 | PC 화면에서 사용할 배경 이미지 경로입니다. |
| 모바일 이미지 경로 | 모바일 화면에서 사용할 배경 이미지 경로입니다. |
| 이미지 위치 | 이미지 초점 위치입니다. 예: `center center`, `50% 40%` |
| 모바일 이미지 위치 | 모바일에서의 이미지 초점 위치입니다. |

수정 순서:

1. 상단 메뉴에서 `홈`을 선택합니다.
2. 하부 탭에서 `메인 비주얼`을 선택합니다.
3. 수정할 슬라이드의 문구를 변경합니다.
4. 필요한 경우 이미지 경로를 변경합니다.
5. `저장` 버튼을 누릅니다.
6. 실제 홈페이지 첫 화면을 새로고침하여 확인합니다.

슬라이드 순서 변경:

- `위로` 버튼: 슬라이드를 앞 순서로 이동합니다.
- `아래로` 버튼: 슬라이드를 뒤 순서로 이동합니다.
- `슬라이드 추가` 버튼: 새 슬라이드를 추가합니다.
- `삭제` 버튼: 해당 슬라이드를 삭제합니다.

이미지 경로 입력 예시:

```text
/hero/homepage/shinhan-hq-building.jpg
/hero/homepage/port-cranes-blue-sky.jpg
```

새 이미지를 사용하려면 먼저 운영 서버에 이미지가 등록되어 있어야 합니다. 경로만 임의로 입력하면 이미지가 보이지 않을 수 있습니다.

### 4.2 홈 섹션 문구 수정

홈 화면에는 무역 동향, 소식지, 사무소, 업무 분야 등의 섹션이 있습니다. 하부 탭을 선택하면 각 섹션의 제목, 설명, 버튼 문구를 수정할 수 있습니다.

수정 순서:

1. 수정할 섹션 탭을 선택합니다.
2. `제목`, `요약 문구`, `버튼 문구`를 수정합니다.
3. 영문 화면도 사용하는 경우 `영문 제목`, `영문 요약 문구`, `영문 버튼 문구`도 수정합니다.
4. `저장` 버튼을 누릅니다.
5. 실제 홈페이지에서 한국어/영어 화면을 각각 확인합니다.

## 5. 회사소개, 업무분야, IT, 문의, 채용, 오시는 길 수정

이 메뉴들은 대부분 같은 방식으로 사용합니다.

공통 수정 순서:

1. 상단 메뉴에서 수정할 영역을 선택합니다.
2. 하부 탭이 있으면 수정할 페이지 또는 섹션을 선택합니다.
3. 입력란의 문구를 수정합니다.
4. 오른쪽 또는 아래쪽의 실제 페이지 미리보기를 확인합니다.
5. 문제가 없으면 `저장` 버튼을 누릅니다.
6. 실제 홈페이지 화면에서 다시 확인합니다.

문구 입력 시 주의사항:

- 긴 문장은 줄바꿈이 어색하지 않도록 실제 화면에서 확인합니다.
- 한글 문구만 수정하면 영어 화면에는 기존 문구가 남을 수 있습니다.
- URL 입력란에는 `/services/import-export`처럼 슬래시(`/`)로 시작하는 내부 경로나 `https://...` 외부 주소를 입력합니다.
- 이미지 경로 입력란에는 실제 존재하는 이미지 경로만 입력합니다.

## 6. 구성원 관리

접속 경로:

```text
/admin/members
```

구성원 메뉴에서는 임원진, 전문가, 자문위원 등의 정보를 관리합니다.

### 6.1 기존 구성원 수정

수정 순서:

1. 왼쪽 `구성원 목록`에서 수정할 사람을 선택합니다.
2. 오른쪽 `구성원 편집`에서 이름, 직함, 부서, 전화, 이메일 등을 수정합니다.
3. 사진을 바꾸려면 `프로필 이미지 선택`에서 이미지 파일을 선택합니다.
4. `이미지 업로드` 버튼을 누릅니다.
5. 업로드가 끝나면 `이미지 경로` 입력란에 새 경로가 자동으로 들어갑니다.
6. `구성원 저장` 버튼을 눌러 최종 저장합니다.

중요: 이미지를 업로드한 뒤에도 `구성원 저장`을 눌러야 구성원 정보에 반영됩니다.

### 6.2 새 구성원 등록

등록 순서:

1. 하단의 `새 구성원` 버튼을 누릅니다.
2. 이름, 그룹, 직함, 부서, 전화, 이메일을 입력합니다.
3. 필요한 경우 프로필 이미지를 선택하고 `이미지 업로드`를 누릅니다.
4. `구성원 저장` 버튼을 누릅니다.

그룹 입력 예시:

```text
executive
expert
advisor
featured, executive
```

주요 그룹 의미:

| 그룹 | 의미 |
| --- | --- |
| `executive` | 임원진 페이지에 표시됩니다. |
| `expert` | 전문가 페이지에 표시됩니다. |
| `advisor` | 자문위원으로 표시됩니다. |
| `featured` | 대표 구성원 영역에 표시됩니다. |

여러 그룹에 동시에 표시하려면 쉼표로 구분합니다.

```text
featured, executive
```

### 6.3 구성원 사진 관리

업로드 가능한 파일:

```text
PNG, JPG, WEBP, GIF
```

사진 권장사항:

- 정면 인물 사진을 사용합니다.
- 배경이 너무 복잡하지 않은 사진이 좋습니다.
- 같은 페이지에 표시되는 사진들은 비슷한 비율과 톤으로 맞추면 보기 좋습니다.

이미지 채우기:

| 값 | 설명 |
| --- | --- |
| `cover` | 영역을 꽉 채웁니다. 일부가 잘릴 수 있습니다. |
| `contain` | 이미지 전체가 보이게 표시합니다. 여백이 생길 수 있습니다. |

이미지 위치 예시:

```text
center center
50% 40%
top center
```

### 6.4 구성원 삭제

삭제 순서:

1. 왼쪽 목록에서 삭제할 구성원을 선택합니다.
2. 오른쪽에서 선택한 사람이 맞는지 확인합니다.
3. `구성원 삭제` 버튼을 누릅니다.
4. 실제 홈페이지에서 해당 구성원이 사라졌는지 확인합니다.

삭제는 복구가 어려울 수 있으므로, 잘못 선택하지 않았는지 반드시 확인합니다.

### 6.5 전문가 배치 설정 JSON

구성원 화면 하단에는 `구성원 설정 JSON` 영역이 있습니다. 이 영역은 분야별 전문가 배치나 IT 문의 담당자 연결을 조정하는 고급 설정입니다.

일반 운영자는 이 영역을 직접 수정하지 않는 것을 권장합니다. 잘못 수정하면 전문가 목록 순서나 연결 정보가 깨질 수 있습니다.

수정이 꼭 필요한 경우:

1. 수정 전 전체 내용을 복사해 별도로 보관합니다.
2. 필요한 값만 조심해서 변경합니다.
3. `설정 저장`을 누릅니다.
4. 전문가 페이지와 IT 페이지를 확인합니다.

## 7. 신한 NEWS 관리

접속 경로:

```text
/admin/news/shinhan-news
```

신한 NEWS 관리 화면:

![신한 NEWS 관리 화면](assets/admin-shinhan-news.png)

### 7.1 새 글 작성

작성 순서:

1. 상단 메뉴에서 `소식/자료`를 선택합니다.
2. 하부 메뉴에서 `신한 NEWS`를 선택합니다.
3. `새 글 작성` 버튼을 누릅니다.
4. 구분, 게시일, 작성자, 제목, 요약을 입력합니다.
5. 본문을 입력합니다.
6. 필요한 경우 본문 사진을 넣습니다.
7. `글 저장` 버튼을 누릅니다.
8. 실제 신한 NEWS 페이지에서 글이 보이는지 확인합니다.

입력 예시:

```text
구분: 공지
게시일: 2026.05.18
작성자: 신한관세법인
제목: 관세·무역 주요 업데이트 안내
요약: 주요 관세 및 무역 이슈를 안내드립니다.
```

### 7.2 기존 글 수정

수정 순서:

1. 왼쪽 목록에서 수정할 글을 선택합니다.
2. 오른쪽 입력 영역에서 내용을 수정합니다.
3. `글 저장` 버튼을 누릅니다.
4. 실제 홈페이지에서 변경된 내용을 확인합니다.

### 7.3 글 삭제

삭제 순서:

1. 왼쪽 목록에서 삭제할 글을 선택합니다.
2. 제목과 게시일을 확인합니다.
3. `글 삭제` 버튼을 누릅니다.
4. 실제 목록에서 사라졌는지 확인합니다.

삭제한 글은 다시 등록해야 하므로 신중하게 처리합니다.

## 8. 소식지 관리

접속 경로:

```text
/admin/news/newsletter
```

소식지 관리 화면:

![소식지 관리 화면](assets/admin-newsletter.png)

소식지는 PDF 파일을 업로드해서 등록합니다. 별도의 `preview.zip` 파일은 필요하지 않습니다.

### 8.1 새 소식지 등록

등록 순서:

1. 상단 메뉴에서 `소식/자료`를 선택합니다.
2. 하부 메뉴에서 `소식지`를 선택합니다.
3. `PDF 파일` 영역에서 PDF 파일을 선택하거나 끌어다 놓습니다.
4. `발행월`을 입력합니다. 예: `2026.04`
5. `게시일`을 입력합니다. 예: `2026.04.27`
6. `제목`을 입력합니다.
7. `요약`을 입력합니다.
8. `새 소식지 업로드` 버튼을 누릅니다.
9. 실제 소식지 페이지에서 목록과 PDF 열람이 되는지 확인합니다.

입력 예시:

```text
발행월: 2026.04
게시일: 2026.04.27
제목: 2026년 4월호 소식지
요약: 2026년 4월 주요 관세·무역 이슈를 정리한 소식지입니다.
PDF 파일: newsletter-2026-04.pdf
```

### 8.2 기존 소식지 수정

수정 순서:

1. 왼쪽 목록에서 수정할 소식지를 선택합니다.
2. 제목, 요약, 발행월, 게시일을 수정합니다.
3. 파일을 교체해야 하면 새 PDF를 선택합니다.
4. `수정 저장` 버튼을 누릅니다.
5. 실제 페이지에서 변경 내용을 확인합니다.

### 8.3 소식지 삭제

삭제 순서:

1. 왼쪽 목록에서 삭제할 소식지를 선택합니다.
2. 제목과 발행월을 확인합니다.
3. `소식지 삭제` 버튼을 누릅니다.
4. 실제 소식지 목록에서 사라졌는지 확인합니다.

## 9. 신한 Insights 관리

접속 경로:

```text
/admin/news/shinhan-insights
```

신한 Insights는 관세/국제 통상 관련 전문 글을 등록하는 화면입니다.

### 9.1 새 글 작성

작성 순서:

1. 상단 메뉴에서 `소식/자료`를 선택합니다.
2. 하부 메뉴에서 `신한 Insights`를 선택합니다.
3. `새 글 작성`을 누릅니다.
4. 카테고리, 게시일, 작성자, 제목, 요약, 본문을 입력합니다.
5. 영문 화면도 사용할 경우 영문 제목, 영문 요약, 영문 본문도 입력합니다.
6. `저장` 버튼을 누릅니다.
7. 실제 신한 Insights 목록과 상세 페이지를 확인합니다.

### 9.2 기존 글 수정

수정 순서:

1. 왼쪽 목록에서 수정할 글을 선택합니다.
2. 오른쪽 입력 영역에서 내용을 수정합니다.
3. `저장` 버튼을 누릅니다.
4. 실제 페이지에서 변경 내용을 확인합니다.

### 9.3 관리자 비밀번호 변경

신한 Insights 화면 하단에는 `글쓰기 권한` 영역이 있습니다. 여기서 관리자 비밀번호를 변경할 수 있습니다.

변경 순서:

1. `현재 비밀번호`를 입력합니다.
2. `새 비밀번호`를 입력합니다.
3. `새 비밀번호 확인`에 같은 값을 한 번 더 입력합니다.
4. `비밀번호 변경` 버튼을 누릅니다.
5. 변경 완료 메시지가 나오면 다음 로그인부터 새 비밀번호를 사용합니다.

비밀번호 규칙:

- 새 비밀번호는 8자 이상이어야 합니다.
- 현재 비밀번호가 틀리면 변경되지 않습니다.
- 새 비밀번호와 확인 입력이 다르면 변경되지 않습니다.

## 10. 저장 후 확인 방법

저장 후에는 실제 홈페이지에서 반드시 확인합니다.

확인 순서:

1. 저장 완료 메시지를 확인합니다.
2. `새 탭에서 크게 보기` 버튼이 있으면 눌러 실제 화면을 엽니다.
3. 브라우저 새로고침을 합니다.
4. 한국어/영어 전환이 필요한 페이지는 두 언어 모두 확인합니다.
5. 이미지, PDF, 링크가 제대로 열리는지 확인합니다.

자주 놓치는 항목:

- 영문 문구를 수정하지 않아 영어 화면에 기존 내용이 남아 있는 경우
- 이미지 경로가 잘못되어 이미지가 보이지 않는 경우
- PDF 파일을 선택하지 않고 소식지 등록을 시도한 경우
- 이미지 업로드 후 `구성원 저장`을 누르지 않은 경우
- 수정 후 실제 페이지 새로고침을 하지 않아 이전 내용으로 보이는 경우

## 11. 자주 묻는 질문

### 저장했는데 홈페이지에 바로 안 보여요.

먼저 홈페이지를 새로고침합니다. 그래도 안 보이면 저장 완료 메시지가 나왔는지 확인합니다. 저장 버튼을 누르지 않았거나, 읽기 전용 모드일 수 있습니다.

### 이미지를 바꿨는데 사진이 안 보여요.

이미지 경로가 실제 존재하는지 확인합니다. 구성원 사진은 이미지 업로드 후 `구성원 저장`까지 눌러야 반영됩니다.

### 소식지 등록에 preview.zip이 필요한가요?

아니요. 새 소식지는 PDF 파일만 업로드하면 됩니다.

### 비밀번호를 바꾸면 기존 환경변수 비밀번호도 바뀌나요?

화면에서 비밀번호를 변경하면 운영 저장소에 변경된 비밀번호 정보가 저장됩니다. 이후 로그인은 변경된 비밀번호를 우선 사용합니다.

### 실수로 내용을 잘못 수정했어요.

저장 전이면 `되돌리기`를 누르면 됩니다. 이미 저장했다면 이전 내용을 다시 입력해 저장해야 합니다. 큰 변경 전에는 기존 문구를 별도로 복사해 두는 것이 안전합니다.

### 삭제한 글이나 구성원을 복구할 수 있나요?

관리자 화면에서 바로 복구하는 기능은 없습니다. 삭제 전 반드시 대상이 맞는지 확인하고, 실수로 삭제했다면 운영 담당자에게 문의해야 합니다.

## 12. 운영자 체크리스트

수정 전:

- 수정할 페이지와 항목을 정확히 확인했습니다.
- 기존 문구를 필요하면 따로 복사해 두었습니다.
- 새 이미지나 PDF 파일 이름을 확인했습니다.

수정 중:

- 한글과 영문 문구를 모두 확인했습니다.
- 이미지 또는 PDF 업로드 후 저장 버튼까지 눌렀습니다.
- 삭제 버튼을 누르기 전 대상을 다시 확인했습니다.

수정 후:

- 저장 완료 메시지를 확인했습니다.
- 실제 홈페이지에서 새로고침 후 확인했습니다.
- 모바일 화면에서도 중요한 이미지와 문구가 어색하지 않은지 확인했습니다.
- 링크와 PDF 파일이 정상적으로 열리는지 확인했습니다.
